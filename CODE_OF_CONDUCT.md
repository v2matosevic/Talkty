# Code of conduct

This is a small open source project. The rules are simple.

- Be respectful. Assume good faith. Critique ideas, not people.
- Keep it on topic and constructive. We are all here to make a useful tool better.
- No harassment, discrimination, or personal attacks. None.

This applies to issues, pull requests, discussions, and any other space connected
to the project.

If someone is making the project a worse place to be, email
**matosevic.markom@gmail.com**. Reports are handled privately. Maintainers may edit,
remove, or block content and contributors that break these rules.
