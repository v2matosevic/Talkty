**Talkty: local, private speech to text. Now fully open source.**

Press a hotkey, speak, and your words are on the clipboard. Whisper runs on your
device. No account, no cloud, no telemetry. Optional cloud transcription and a
Prompting mode are available when you want them, both off by default.

- 100% local transcription with optional GPU acceleration (CUDA, Vulkan, CPU)
- Coding vocabulary, clean output, and auto-paste at the cursor
- Cloud transcription and Prompting via OpenRouter, opt-in, key encrypted on device
- MIT licensed

See the README for install and the CHANGELOG for the full history.
